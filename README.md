# my-godot

Godot 学习项目仓库，基于 [官方文档](https://docs.godotengine.org/zh-cn) 驱动，包含多个子项目。

## 子项目

- [2d-1](./2d-1/) - 2D 游戏项目

## 环境安装

```bash
scoop install godot
```

### Godot 引擎

- [Godot 下载](https://godotengine.org/download/archive)

### .NET 运行时

Godot 某些功能需要 .NET 运行时，下载地址：

- [.NET 下载](https://dotnet.microsoft.com/zh-cn/download/dotnet)
